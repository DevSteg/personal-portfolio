Projects

Abigail Flower Art

Project Info

This project is for a Art and pottery store called Abigail Flower. The client wanted a site where they could showcase their products for sale and allow the users to purhcase items. The website was built using Django and was deployed with Heroku and is using a POSTgres SQL database. Stripe has been used to control the payment system and security. AWS S3 Bucket has been used for image hosting.

Technologies Used
  • Django
  • HTML5
  • CSS3
  • Bootstrap
  • Javascript
  • AWS (S3 Bucket)
  • Stripe

ToDo Dashboard

Project Info

This project was inspired by the dashboard extention for Chrome called Momentum. The web app is a todo list daashboard, built with javescript. The todo list makes use of the local storage. The app also asks for location data as it has a weather section that will give the user the local weather in a widget. The app uses the open weather map API for the weather section.

Technologies Used
  • HTML5
  • CSS3
  • Bootstrap
  • Javascript
  • Open Weather Map API

Film Review

Project Info

This project is a Web application for film fans to share their own reviews about the films on the site. The films are added and managed by the site users. The app utilizies MongoDB for its database and was built using Flask, Materialize was used for the styling of the site.
